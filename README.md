# 📝 Lista de Tarefas em Python

Sistema de gerenciamento de tarefas desenvolvido em Python com persistência em JSON.

## 🚀 Funcionalidades

- Adicionar tarefas com prioridade
- Listar tarefas organizadas
- Marcar como concluída
- Excluir tarefas
- Filtrar por prioridade e status
- Buscar tarefas por nome
- Estatísticas com porcentagem de conclusão

## 🛠 Tecnologias

- Python
- JSON

## ▶️ Como executar

```bash
python lista_de_tarefas.py
